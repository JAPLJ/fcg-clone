カード効果に使用されている用語を解説します。

1. 凡例

次セクションでは次のような形式で効果説明の用語を紹介していきます。

効果
[発動条件] ～～～解説～～～

効果: 実際にカードに効果説明として書かれている用語です。色の名前を color、整数値を n で表します。

[発動条件]: 効果の発動条件です。出現時ならモンスターの召喚時か呪文の詠唱時に効果が発動します。ターン終了時なら、モンスターが場に召喚されている状態か、そのターンに呪文が詠唱されている場合のターン終了時に効果が発動します。

2. 用語解説: 毒・リジェネ関連

毒 n
[出現時] 自モンスターに毒を n 与える。
服毒 n
[出現時] 自プレイヤーに毒を n 与える。
毒付与 n
[出現時] 相手プレイヤーに毒を n 与える。
猛毒付与 n
[ターン終了時] 相手プレイヤーに毒を n 与える。
祝福 n
[出現時] 自モンスターにリジェネを n 与える。
リジェネ n
[出現時] 自プレイヤーにリジェネを n 与える。

3. 用語解説: エネルギー関連

常時 color エネルギー消費 n
[ターン終了時] color 色のエネルギーを n 消費する。所持エネルギーが n に満たない場合は 0 になる。
color エネルギー消費 n
[出現時] color 色のエネルギーを n 消費する。所持エネルギーが n に満たない場合は 0 になる。
color ジェネレーター獲得 n
[出現時] color 色のジェネレーターを n 個得る。
color エネルギー獲得 n
[出現時] color 色のエネルギーを n 個得る。

4. 用語解説: ダメージ関連

即時ダメージ n
[出現時] 相手プレイヤーにダメージを n 与える。
即時自傷ダメージ n
[出現時] 自プレイヤーにダメージを n 与える。
即時モンスターダメージ n
[出現時] 相手モンスターにダメージを n 与える。
即時モンスター自傷ダメージ n
[出現時] 自モンスターにダメージを n 与える。
追加ダメージ n
[ターン終了時] 相手プレイヤーにダメージを n 与える。

5. 用語解説: モンスター除去・操作関連

除去
[出現時] 相手がモンスターを召喚していれば、それを破壊する。
HP 吸収
[出現時] 相手がモンスターを召喚していれば、その HP と同じ分だけ自プレイヤーは HP を回復する。
生贄
[出現時] 自分がモンスターを召喚していれば、それを破壊する。破壊されたモンスターの HP・攻撃力・防御力・リジェネ値と同じ分だけ、自プレイヤーは HP・攻撃力・防御力・リジェネ値を得る。
凍結 n
[出現時] 相手モンスターを n ターン凍結状態にする (すでに凍結されていれば、凍結ターンを n 延長する)。
解凍
[出現時] 自モンスターが凍結状態にあるならば、それを解除する。

6. 用語解説: ステータス変化関連

攻撃力UP n / 攻撃力DOWN n
[出現時] 自プレイヤーの攻撃力が n 増える / n 減る。
防御力UP n / 防御力DOWN n
[出現時] 自プレイヤーの防御力が n 増える / n 減る。
敵攻撃力UP n / 敵攻撃力DOWN n
[出現時] 相手プレイヤーの攻撃力が n 増える / n 減る。
敵防御力UP n / 敵防御力DOWN n
[出現時] 相手プレイヤーの防御力が n 増える / n 減る。
モンスター攻撃力UP n / モンスター攻撃力DOWN n
[出現時] 自モンスターの攻撃力が n 増える / n 減る。
モンスター防御力UP n / モンスター防御力DOWN n
[出現時] 自モンスターの防御力が n 増える / n 減る。
回復 n
[出現時] 自プレイヤーの HP を n 回復する。

7. 用語解説: その他

ドロー n
[出現時] 山札からカードを n 枚ドローする。既に手札を 7 - n 枚以上持っているときは、手札が 6 枚になるまでドローする。
